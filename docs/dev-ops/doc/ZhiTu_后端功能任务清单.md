## 一、登录/注册模块

### 1.1 注册流程
- [x] 界面一：基础账号信息（手机号/邮箱/密码/昵称）
- [x] 界面二：简历与基础画像
  - [x] 必填项：专业、年级、base（最多3个）
  - [x] 补充项：意向岗位、绩点、政治面貌、荣誉证书、项目经历、校园经历、技能、语言、自我评价、实习经历
  - [ ] 简历上传解析（PDF/WORD/图片 → AI 提取信息后可修改）
  - [ ] 无简历时的快速制作引导
- [x] 界面三：职场偏好权重分配（点数分配模式）
  - [x] 维度：薪资水平、福利保障、稳定性、生活成本、时长/加班、晋升空间、兴趣、成长机会、技术门槛、涨薪空间
- [x] 界面四：软实力自评（五级制）
  - [x] 维度：语言表达、人际交往、抗压、资源协调、创新、变通、学习能力、实习能力
- [x] 界面五：心理/职业性格测试（可选）
  - [x] MBTI、霍兰德、Big Five

### 1.2 登录
- [x] 账号登录（手机号/邮箱 + 密码）
- [ ] 第三方授权登录：QQ
- [ ] 第三方授权登录：微信
